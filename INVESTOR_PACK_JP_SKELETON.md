# Investor Pack Skeleton (JP Parent Audience)

## 1. Problem
- Students with high grades still apply without direction.
- Fragmented workflow: academics, activities, admissions strategy are disconnected.

## 2. Solution
- One ecosystem: **Direction -> Weekly Execution -> Outcome Tracking**
- Niche-first: IB-focused planning and guidance.

## 3. Product (current)
- Intake + profile engine
- Activity planner
- Question bank + practice scoring
- Notes + flashcards + progress

## 4. Why now
- AI lowers advisory cost while increasing personalization.
- Families seek strategic guidance beyond tutoring-only services.

## 5. Pilot plan (30 days)
- 20–30 users from immediate network
- 3 KPIs: activation, WAU retention, outcome proxy
- Week-2 monetization experiment

## 6. Use of funds (90 days)
- Infra and API reliability
- Curated opportunity database pipeline
- Content QA + legal-safe sourcing
- Minimal operator bandwidth for support

## 7. Risk control
- Governance docs before scale
- IP assignment / licensing clarity
- Data privacy + auditability
- Controlled burn and monthly reporting

## 8. Milestones
- Day 30: pilot KPI baseline
- Day 60: improve retention + conversion
- Day 90: expand distribution and close follow-on commitments

## 9. Funding instrument options
- Founder loan
- SAFE/convertible
- priced equity (post-traction)

## 10. Ask
- Short runway to execute 90-day plan with KPI-based accountability.
