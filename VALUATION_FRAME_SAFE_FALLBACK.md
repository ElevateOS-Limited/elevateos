# Valuation Frame (Pre-incorporation / Early Pilot)

## Principle
Do not anchor valuation to generic averages alone.
Anchor to:
1) pilot revenue potential,
2) execution risk discount,
3) milestone-based de-risking.

## Suggested framing language
- Current discussion is a **pre-money working range** for planning, not a final legal valuation.
- Final valuation can be set after pilot KPI evidence.

## KPI gates before hard pricing
- Activation rate
- Week-1 / Week-4 retention
- Outcome proxy improvement
- Early paid conversion signals

## SAFE fallback
If valuation confidence is low, use a simple SAFE/convertible note now and defer final pricing to the next qualified round.

## Investor communication rule
- No guaranteed return language.
- Use risk-adjusted scenarios (optimistic / base / pessimistic).
- Tie funding to milestones and reporting cadence.
